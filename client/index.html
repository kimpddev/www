  • 네이버 실시간 검색어
    확장 프로그램
    지원 브라우저 : 크롬, 사파리
    크롬 또는 사파리에서 네이버의 실시간 급상승 검색어를 간편하게 확인할 수 있습니다.

    - 간단 검색 기능
    - 선택 단어 검색 기능 (마우스 우클릭 메뉴)
    - 실시간 업데이트
